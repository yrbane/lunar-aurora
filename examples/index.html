Code & Syntaxe

Variables CSS pour la coloration syntaxique des blocs de code.

Bloc de code

const ThemeSwitcher = { storageKey: 'lunar-theme', setTheme(name) { document.documentElement .setAttribute('data-theme', name); localStorage.setItem( this.storageKey, name ); } };
